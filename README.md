# Mobile Banking App

Aplicacion movil de banca desarrollada con Flutter como proyecto de aprendizaje.

## Descripcion

Este proyecto simula una app bancaria con pantallas de autenticacion, home, historial, transferencias y configuracion.

## Tecnologias

- Flutter
- Dart
- go_router (navegacion)
- dio (cliente HTTP reusable)

## Estructura General

- lib/core: utilidades compartidas (assets, red, etc.)
- lib/features: modulos por funcionalidad (login, home, transfers, history, settings)
- lib/widgets: widgets globales reutilizables

## Requisitos

- Flutter SDK instalado
- Dart SDK (incluido con Flutter)

## Como ejecutar el proyecto

1. Clonar el repositorio.
2. Instalar dependencias:

```bash
flutter pub get
```

3. Ejecutar la app:

```bash
flutter run
```
